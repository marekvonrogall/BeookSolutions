# Beook Solutions

## Purpose
This application is used to add a new toolbar item to "[Beook](https://beook.ch/)", a digital learning platform. When you press the button, the sample solution for the current task is displayed.
This is useful if you get stuck on a task or are studying for an exam and therefore want to look at the solutions. However, I don't recommend using this tool to skip the learning process by copying from the solutions.

## Requirements

In order to use this application you need [install Beook](https://beook.ch/download) and log in to your account.

## Setup

You can find the newest release on the *[releases page](https://github.com/marekvonrogall/BeookSolutions/releases)*.

**Installation:**
- Download and extract the .zip file.
- Run the `setup.exe` and follow the instructions.

Running the application for the first time requires the user to perform some steps in order to complete the setup. This is necessary in order to grab some crucial information from a log file that Beook creates on startup.

> [!IMPORTANT]  
> Make sure to keep the default workspace directory within Beook, so the application knows where to get the log file from.
> The default workspace directory is `C:\Users\%USERNAME%\AppData\Roaming\ionesoft\beook`.
